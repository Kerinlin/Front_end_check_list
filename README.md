# 前端知识点

## 为什么开这个仓库?
1. 希望能系统的总结知识点，巩固夯实自己的基础;
2. 希望能在以后面试过程中更加如鱼得水;
3. 希望能随时记录学习.

## 怎么添加文章？
1. 在相应的板块添加文章;
2. index.md引用文章.

## 文章规范
1. 符合md语法;
2. 新增文章文件名必须简洁明了,不做标题党;
3. 文章内容包含标题，解释两部分,解释部分必须折叠(折叠的目的是让读者能先去自己探索，然后再看看别人的解释);
4. 添加文档，更新文档，提交代码时需要描述关键字(Add,Update)
